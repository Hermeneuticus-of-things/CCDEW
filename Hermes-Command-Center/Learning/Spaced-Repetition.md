# 🔄 Spaced Repetition System

> **Sistem de repetare spațiată pentru memoria pe termen lung**
> *„Repetiția este mama învățăturii"*

---

## 🧠 Cum Funcționează

Repetiția spațiată (SRS) este o tehnică de învățare care programează review-uri la intervale crescătoare:

```
Învățare → Review la 1 zi → Review la 3 zile → Review la 7 zile → Review la 30 zile → Review la 90 zile
```

## 🔧 Implementare în Obsidian

### Fără Plugin (Manual)
1. Adaugă data următorului review în frontmatter
2. Folosește `[[ ]]` pentru a naviga la carduri
3. Review zilnic/săptămânal conform calendarului

### Cu Plugin (Recomandat)
Instalează **Obsidian Spaced Repetition** plugin:
- Adaugă `<!-- flashcard -->` în note
- Sau folosește `::` pentru carduri inline
- Plugin-ul gestionează automat intervalele
