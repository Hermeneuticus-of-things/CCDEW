How It Works

Three-Loop Learning System

Three nested feedback loops, each at a different cadence, driving the system toward 95% convergence.

Loop 1 — Execution

~s

Task in → Enneagram Core selects best node → Pipeline divergent/convergent → Record to SAFLA + episodic memory

hermes-enneagram-core.py ccdew-pipeline.py

Loop 2 — Self-Train

30s

Pick weakest node → generate task → run cycle → measure convergence delta → repeat until 95%

hermes-self-train.py hermes-convergence.py

Loop 3 — Consolidation

10 ep

Every 10 episodes: extract patterns → techniques → attitudes → principles. Reverse mirror to separate channel.

hermes-memory.py

Under The Hood

Infrastructure

Each component has a single responsibility. The bridge eliminates shared-file race conditions entirely.

HTTP Bridge

127.0.0.1:18777

Single Python process serves all state. Others read/write via HTTP. Zero race conditions.

SAFLA State

Written exclusively by pipeline.py

9 nodes tracked. 187 episodes. 100% single-writer — no data loss.

MCP Servers

4 active servers. Native http.request

No shell exec, no curl, no injection vectors.

Inner Observer

Metacognitive daemon

Writes consciousness.jsonl (append-only). Tracks confidence, flow, confusion, satisfaction.

# Bridge endpoints GET /bridge # read current bridge state GET /safla # read SAFLA state POST /safla # update SAFLA (single writer) POST /log # centralized logging GET /core/select?task= # Enneagram node selection POST /core/outcome # learn from outcome GET /convergence # dashboard JSON GET /health # heartbeat
05 / 07

Why CCDEW

Problems Solved

↗
LLMs never improve with use

Convergence engine tracks every node. Self-train daemon drives toward 95%. The system measurably improves.

🧠
One-size-fits-all reasoning

9 Enneagram nodes with unique system prompts, task transforms, and per-node episodic memory. The right persona for every task.

⚡
Race conditions in state files

HTTP bridge with single writer. Others read/write via HTTP POST/GET. Zero data corruption.

🛡
Failure cascades crash everything

Non-fatal pipeline. Each sub-step wrapped in try/except. One failure never takes down the system.

📝
Memory polluted by reflections

Reverse mirror writes to mirror_inverse.jsonl — separate from episodic. Clean separation.

🔒
Shell injection in MCP tools

Native http.request in Node.js. No curl, no exec, no shell injection.
