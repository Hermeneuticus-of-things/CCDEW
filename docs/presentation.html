What's wrong with AI agents today

Four Problems CCDEW Solves

Every AI system today struggles with the same four issues. CCDEW was built from day one to solve all of them.

✕

LLMs never improve with use

Same knowledge today = same knowledge tomorrow. Zero growth between sessions. No learning curve. No compounding improvement.

→ Convergent Self-Training : 73.4% and climbing
✕

One-size-fits-all reasoning

Every task gets the same generic persona. No specialized perspective. A code review uses the same approach as a creative writing task.

→ 9 Enneagram Nodes : best persona per task
✕

No measurable quality

No dashboard. No metric. No way to tell if the system is improving or degrading. Blind deployment.

→ Convergence Score : live metric, 0–100%
✕

Fragile and insecure

One failure crashes everything. Shell injection vectors in MCP. Race conditions in shared state files.

→ HTTP Bridge : no race conditions, no exec

How It Works

Three-Loop Learning System

Three nested feedback loops, each at a different cadence, driving the system toward 95% convergence.

Loop 1 — Execution

≈s

Task in → Enneagram Core selects best node → Pipeline divergent/convergent → Record to SAFLA + episodic memory

hermes-enneagram-core.py ccdew-pipeline.py

Loop 2 — Self-Train

30s

Daemon selects the weakest node → generates a training task → runs full cycle → measures convergence improvement

hermes-self-train.py hermes-convergence.py

Loop 3 — Consolidation

10 ep

Every 10 episodes: extract patterns → techniques → attitudes → principles. Reverse mirror to separate channel.

hermes-memory.py

When to use CCDEW

Use Cases

CCDEW fits best when you need an AI system that actively improves, produces diverse perspectives, and gives you hard metrics.

↗

Self-Improving AI Assistant

Deploy an agent that gets measurably better every day. No manual retuning. No human oversight. The daemon trains the weakest node autonomously.

🧠

Multi-Perspective Analysis

Route a problem through 9 cognitive personas. Get 9 different perspectives from 9 specialized Enneagram types. Converge into one synthetic verdict.

⚙

Quality Gate Automation

Pre-commit checks, pre-push gates, code reviews, risk assessments. Every step routed through the best-suited cognitive node for maximum accuracy.

📚

Research & Learning System

Memory pyramid auto-consolidates everything learned. Raw actions (N1) → abstract principles (N6). No knowledge is ever lost between sessions.
